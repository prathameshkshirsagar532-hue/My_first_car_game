# 🏎️ Simple Canvas Car Game

Welcome to my first web-based 2D Car Game! Built completely from scratch using **HTML5 Canvas** and **Vanilla JavaScript**. 

This project was developed directly from a my device and deployed live using **GitHub Pages**! 😎

---

## 🎮 Live Demo

---

## 🚀 Features
- 📱 **Mobile Friendly:** Fully responsive touch buttons (`◀` and `▶`) for Android/iOS users.
- 💻 **PC Controls:** Smooth movement using Left and Right Arrow keys.
- ⚡ **Dynamic Difficulty:** The game gets faster and harder as your score increases!
- 🎨 **Pure Canvas Graphics:** Rendered instantly using pure code shapes—no external image assets required (zero loading lag).
- 🔄 **Instant Restart:** Just tap the screen after a Game Over to start racing again.

---

## 🛠️ Tech Stack Used
- **Frontend:** HTML5
- **Styling:** CSS3 (Flexbox & responsive design)
- **Game Engine:** JavaScript (Canvas API & `requestAnimationFrame`)
- **Hosting:** GitHub Pages

---

## 📖 How to Play

### On Computer (PC):
1. Use the **Left Arrow Key** to move left.
2. Use the **Right Arrow Key** to move right.
3. Dodge the **Red Obstacles** to keep scoring!

### On Mobile / Tablet:
1. Tap and hold the **◀ button** to move left.
2. Tap and hold the **▶ button** to move right.
3. If you crash, simply **Tap the game screen** to restart.
